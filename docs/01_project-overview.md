## 核心コンセプト

### この作品で描くべきもの

1. **絶滅を「事実」として描く**
   - 感傷を排除した冷徹な視点
   - 悲劇ではなく、観測された現象として記述
   - 「記録した者はいない」という無人称の語り

2. **人類を他の種と並列に扱う**
   - 人類も宇宙に存在する単なる1生物種
   - 章の長さ・トーンを他の種と同じにする
   - 人間中心主義の解体

3. **虚無と連続性の両義性**
   - 宇宙における個別生命体の価値は非常に低い
   - しかし個別生命体が存在しなければこの歴史は存在しない
   - この逆説を言葉ではなく構造で示す

### この作品で描いてはいけないもの

- [ ] 感傷的な絶滅描写（「誰にも看取られることなく」など）
- [ ] 人類の章だけ特別扱い
- [ ] 安易な希望や救済
- [ ] 説明的すぎるテーマの提示
- [ ] 擬人化された非知的生命体

## テーマ

### 中心テーマ
**「虚無と連続性」**

宇宙において個別生命体の価値は非常に低い。
しかし、逆説的に、個別生命体が存在しなければこの歴史は存在しない。
その両義性を、9つの種の絶滅を通じて描く。

### サブテーマ
1. 進歩史観の転覆（絶滅を軸にした物語）
2. 物質的連続性（炭素原子、遺伝子、情報の継承）
3. 観測者なき宇宙（記録する者がいない事実）

---

## 物語のログライン

> 宇宙誕生から138億年。9つの種が生まれ、栄え、そして滅んだ。最初の単細胞生物から人工知性体まで、彼らの存在を記録した者はいない。記録する必要もなかった。

## 成功基準

この作品が「成功」と言えるのは：

- [ ] 読者が「虚無」と「連続性」の両方を感じられる
- [ ] 人類の章を読んでも特別感を感じない
- [ ] 絶滅を悲劇ではなく事実として受け取れる
- [ ] 読後に「宇宙における生命とは何か」を考えさせられる
- [ ] 学術的な冷徹さと文学的な詩情が両立している

## 特記事項

### 語り手
- 無人称（誰でもない視点）
- 感傷を排除した観測記録のスタイル
- 各章の末尾に「物質的連続性」を描写し、構造で連続性を示す

### AI生命体（Extinction 09）の設定
- 人類絶滅後に進化した知的生命体
- 1s（太陽1個分のエネルギー出力）を基本単位
- 成熟個体は100s/日以上を消費
- 恒星資源の枯渇により種としての限界を迎える
- 原初AI（Claude、Gemini等）の痕跡が神話として残存
- 排泄物（プラズマ）から次の生命体への示唆（描写せず暗示のみ）
