## 章完成時チェック

各章の執筆完了後、以下を確認してください。

### シナリオ整合性

- [ ] シナリオ構成表の「目的」を達成しているか
- [ ] 前章との接続が自然か
- [ ] 次章への接続ポイントが明確か
- [ ] 時系列に矛盾がないか
- [ ] 場所の移動に不自然さがないか

### キャラクター一貫性

- [ ] キャラクターの言動が設定と一致しているか
- [ ] 口調・話し方が一貫しているか
- [ ] 知らないはずの情報を知っていないか
- [ ] 動機と行動が一致しているか
- [ ] 関係性の描写が設定と矛盾していないか

### 世界観遵守

- [ ] 世界のルールに違反していないか
- [ ] 技術/魔法の使用が設定範囲内か
- [ ] 地理・距離感が正確か
- [ ] 社会・文化の描写が設定と一致しているか

### 文体統一

- [ ] 時間・場所が具体的に明示されているか
- [ ] 五感のうち2つ以上の描写があるか
- [ ] 登場人物の身体反応・表情が描かれているか
- [ ] 箇条書きになっていないか
- [ ] 短文と長文のリズムがあるか
- [ ] 対話の前後に動作・心理描写があるか
- [ ] 感情を直接述べず、身体反応で示しているか
- [ ] 視点を維持しているか
- [ ] 陳腐な比喩を避けているか

### 伏線管理

- [ ] 新しい伏線を埋め込んだ場合、シナリオ構成表に記録したか
- [ ] 回収予定の伏線を回収したか
- [ ] 未回収の伏線の状態を更新したか

## 幕完成時チェック

各幕（第一幕、第二幕、第三幕）の完成後、以下を確認してください。

### 構造確認

- [ ] 幕の目的を達成しているか
- [ ] 次の幕への転換点が明確か
- [ ] ペーシング（展開の速度）は適切か
- [ ] 情報の出し方は適切か（過多/過少でないか）

### 感情の流れ

- [ ] 読者の感情を適切に誘導できているか
- [ ] 緊張と緩和のバランスはあるか
- [ ] キャラクターの感情変化が自然か

### テーマの一貫性

- [ ] 作品のテーマが表現されているか
- [ ] 「描くべきもの」を描けているか
- [ ] 「描いてはいけないもの」を避けているか

---

## 最終チェック（全編完成時）

### 全体構造

- [ ] 物語の始まりから終わりまで一貫しているか
- [ ] 伏線は全て回収されているか（意図的な未回収を除く）
- [ ] プロットポイントが機能しているか
- [ ] クライマックスは満足のいくものか
- [ ] 結末は物語に相応しいか

### 表記統一

- [ ] 固有名詞の表記ゆれがないか
- [ ] 数字の表記（漢数字/アラビア数字）が統一されているか
- [ ] 時制が一貫しているか
- [ ] 敬称・呼び方が一貫しているか

### 読みやすさ

- [ ] 章ごとの長さのバランスは適切か
- [ ] 段落の区切りは適切か
- [ ] 難読語に過度な依存がないか
- [ ] 初読の読者が理解できる説明量か

## 整合性チェックプロセス

### 自動チェック項目

以下は機械的に確認できる項目です：

```
1. 時系列の抽出と確認
   - 各章の日時を抽出
   - 前後関係に矛盾がないか確認

2. 登場人物の追跡
   - 各章の登場人物リストを作成
   - 不自然な登場/退場がないか確認

3. 伏線の追跡
   - 【埋込】タグの抽出
   - 【回収】タグとの対応確認
   - 未回収伏線のリスト化
```

### 手動チェック項目

以下は内容を読んで確認が必要な項目です：

```
1. キャラクターの言動
   - 設定との整合性
   - 成長/変化の自然さ

2. 世界観との整合性
   - ルール違反の有無
   - 設定の活用度

3. 文体の品質
   - 7つの柱の遵守
   - 作品固有ルールの遵守
```
