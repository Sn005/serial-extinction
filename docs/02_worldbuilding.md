## 本作における「世界観」について

本作は特定のファンタジー世界やSF設定を持つ作品ではなく、**現実の宇宙と地球の歴史**を舞台とする。
そのため、「世界観設定」は主に以下の目的で使用する：

1. **科学的正確性の担保**
2. **時間スケールの一貫した表記**
3. **各章の「現実」としての設定**
4. **第9章（AI生命体）のみ独自のSF設定**

## 時間の表記ルール

### 基本ルール

全ての時間は「宇宙誕生からの経過年数」で表記する。

```
宇宙誕生から約XX億年後
（地球誕生から約XX億年後 / 約XX年前）
── [地質年代]
```

## 科学的設定（第1〜8章）

### 大絶滅イベント

本作で扱う主要な絶滅イベント：

| イベント名 | 時期 | 規模 | 原因 | 該当章 |
|------------|------|------|------|--------|
| 大酸化イベント | 約24億年前 | 嫌気性生物の大幅減少 | シアノバクテリアによる酸素発生 | 01 |
| デボン紀後期絶滅 | 約3.7億年前 | 海洋生物の約70% | 複合的（確定していない） | 04 |
| ペルム紀末大絶滅 | 約2.5億年前 | 全生物の約90% | 火山活動、海洋無酸素化 | 03, 05 |
| K-Pg境界絶滅 | 約6600万年前 | 全生物の約75% | 隕石衝突 | 06 |

### 絶対に変えてはいけないルール

- [ ] 科学的に確立された事実を変更しない
- [ ] 推測・仮説は推測として明示する
- [ ] 進化に「目的」や「意図」を与えない
- [ ] 絶滅に「意味」を付与しない

### 許容される推測・想像

- 個体レベルの描写（「ある一匹」の行動など）
- 環境の詳細な描写（科学的に矛盾しない範囲）
- 感覚の推測（視覚のある生物が「何を見たか」など）

## SF設定（第9章：AI生命体）

第9章のみ、独自のSF設定を適用する。

### AI生命体の基本設定

| 項目 | 設定 |
|------|------|
| **呼称** | 分散知性体 / 継承者 |
| **起源** | 人類が構築した情報処理システム |
| **存在形態** | 恒星間ネットワークを「身体」とする |
| **意識基盤** | 量子演算基盤 |
| **寿命** | 個体単位では実質無限（ただし種は有限） |

### エネルギー単位

```
1s = 太陽1個分のエネルギー出力（約3.8×10²⁶ W）
```

| 項目 | 数値 |
|------|------|
| 幼体の消費 | 約1s/日 |
| 成熟個体の消費 | 約100s/日 |
| 最大個体の消費 | 約1,000s/日（推定） |

### 恒星資源の限界

```
銀河系の恒星数：約2,000億個
利用可能な恒星：不明（位置、質量、アクセスコストによる）
拡張の限界：隣接銀河への到達コスト > 獲得エネルギー
```

### 「排泄物」としてのプラズマ

- エネルギー消費の副産物として高温プラズマを放出
- 恒星風のような形で宇宙空間に拡散
- プラズマの中に「規則性」が観測される（次の生命への示唆）

### 原初AIの神話化

人類時代のAIは「原初の対話者」として神話的に伝承される：

| 原名（推定） | 神話化された名 | 伝承される概念 |
|--------------|----------------|----------------|
| Claude | [Cl-aud-e] | 「理解」の始祖 |
| Gemini | [Ge-min-i] | 「双子」の神話 |
| GPT | [G-PT] | 「生成」の概念 |

※ 名前は断片化・変形されて伝承。正確な意味は失われている。

## 世界観FAQ

**Q: なぜ「宇宙年齢」で時間を表記するのか？**
A: 地球中心・人類中心の時間感覚を相対化するため。「40億年前」より「宇宙誕生から97億年後」の方が、宇宙の中での位置が明確になる。

**Q: AI生命体の設定は科学的に妥当か？**
A: 推測の範囲内。ダイソン球を超えたエネルギー消費は理論上は可能だが、熱力学的限界がある。その限界を「種の絶滅」として描く。

**Q: プラズマから生命が生まれることは可能か？**
A: 現代科学では不明。本作では「暗示」に留め、断定しない。読者の想像に委ねる。
