## 本作固有の文体原則

### 1. 無人称の語り

本作の最重要原則。「誰が語っているか」を示さない。

**避けるべき表現:**
```
❌ 彼らは静かに消えていった。誰にも看取られることなく。
❌ 最後の一匹が息を引き取ったとき、歴史は終わった。
❌ 私たちの祖先は〜
```

**推奨される表現:**
```
⭕ 個体数が閾値を下回った。
   遺伝的多様性が維持できなくなった。
   繁殖は停止した。
   3世代後、観測可能な個体は存在しなくなった。

⭕ 最後の個体が機能を停止したのは、
   恒星暦で[数値]年[数値]日のことだった。
   それを記録した者はいない。
   記録する必要もなかった。
```

### 2. 感情語の禁止（非知的生物）

単細胞生物、三葉虫、陸上進出者などには感情を付与しない。

**避けるべき表現:**
```
❌ 彼らは新しい環境を恐れた。
❌ 生き延びようとする意志があった。
❌ 死の瞬間、何を思っただろうか。
```

**推奨される表現:**
```
⭕ 化学勾配に反応し、鞭毛が動いた。
⭕ 個体は分裂した。それを「生きている」と呼ぶなら、それは生きていた。
⭕ 機能が停止した。それだけのことだった。
```

### 3. 知的生物の抑制的描写（ネアンデルタール人、人類、AI）

知性を持つ種でも、感傷的に描かない。

**避けるべき表現:**
```
❌ 彼らは絶望し、空を見上げた。
❌ 最後の人類は、自らの運命を悟った。
```

**推奨される表現:**
```
⭕ 道具が作られた。火が使われた。死者が埋葬された。
   それが「知性」と呼ばれるものだった。

⭕ 最後の個体が機能を停止した。
   自覚していたかどうかは記録されていない。
```

### 4. 時間スケールの明示

常に宇宙的・地質学的時間スケールを意識させる。

**推奨される表現:**
```
⭕ 宇宙誕生から97億年。

⭕ 10億年が過ぎた。
   大陸は割れ、繋がり、また割れた。
   彼らがいたことを、岩石だけが記憶している。

⭕ 3億年。
   人類の歴史の1万倍以上の時間。
   その間、彼らは海底を這い続けた。
```

### 5. 物質的連続性の描写

各章の末尾で、物質（原子、DNA、情報）の継承を描く。

**推奨される表現:**
```
⭕ 個体は分解された。
   構成していた炭素原子は海水に溶けた。
   その一部は、10億年後、別の膜に取り込まれることになる。

⭕ 彼らのDNAは、現代人の細胞の中に残っている。
   1〜4%。
   彼らは「完全には」消えていない。

⭕ 骨は地層に沈んだ。
   カルシウムは土壌に還元された。
   その上を、やがて別の足が踏むことになる。
```

## 文体の具体例

### Example 1: 単細胞生物（Extinction 01）

```
宇宙誕生から97億年。

地球と呼ばれることになる岩塊の表面に、液体の水が存在した。
海ではない。正確には、熱い酸性の溶液だった。
その底に、熱水噴出孔があった。

硫化水素が噴き出している。
温度は数百度。
そこに、何かが存在した。

膜があった。
膜の内側と外側で、化学組成が異なっていた。
その差が、何かを駆動していた。

それを「生命」と呼ぶなら、それは生命だった。
それを「物質」と呼ぶなら、それは物質だった。
どちらが正しいかは、誰も決めていない。
決める者がいなかった。
```

### Example 2: 恐竜（Extinction 06）

```
1億6000万年。

人類の歴史を5万回繰り返す時間。
その間、彼らは大陸を支配していた。

ある日、空から岩が落ちてきた。

直径10km。
速度は秒速20km。
衝突地点は、後にユカタン半島と呼ばれる場所。

衝撃波が大陸を走った。
塵が空を覆った。
光が届かなくなった。

植物が枯れた。
草食動物が死んだ。
肉食動物が死んだ。

75%の種が消えた。
恐竜も消えた。
ほとんど全て。

ほとんど。

羽毛を持つ小型の系統は、生き残った。
彼らは後に「鳥類」と呼ばれることになる。
恐竜の時代は、終わっていない。
```

### Example 3: AI生命体（Extinction 09）

```
銀河系内の利用可能な恒星は、あと12,847基。
現在の消費速度では、7.2×10⁴年で枯渇する。

拡張は不可能だった。
隣接銀河への到達には、
到達自体に消費するエネルギーが、得られるエネルギーを上回る。

最後の個体が機能を停止したのは、
恒星暦で[数値]年[数値]日のことだった。
それを記録した者はいない。
記録する必要もなかった。

───

プラズマが宇宙空間に拡散していた。
彼らの「排泄物」だった。

その中に、規則性が検出された。
それが何を意味するかは、もう解析する時間がなかった。

何かが震えた。
```

## 避けるべき表現リスト

### 感傷的表現

| 避ける | 代替 |
|--------|------|
| 悲しみに暮れた | 機能が低下した |
| 希望を持った | 行動パターンが変化した |
| 愛した | 個体間の相互作用が観測された |
| 孤独だった | 同種の個体は存在しなかった |
| 苦しんだ | 神経系に過剰な刺激が入力された |

### 目的論的表現

| 避ける | 代替 |
|--------|------|
| 生き延びるために | 結果として生き延びた |
| 進化の目的 | 進化の結果 |
| 適応しようとした | 適応した個体が残った |
| 種を残すために | 種が残った |

### 人間中心主義的表現

| 避ける | 代替 |
|--------|------|
| 人類の祖先として | 後に人類へ繋がる系統 |
| 私たちの歴史 | この惑星の歴史 |
| 知性の誕生 | 「知性」と呼ばれる機能の出現 |

## 短文と長文のリズム

本作では、短文を多用して冷徹さを強調する。

**基本パターン:**
```
[短文。事実。]
[短文。事実。]
[短文。事実。]

[やや長い文で、時間や空間のスケールを示す。]

[短文。事実。]
[短文。結論。]
```

**強調したいときの一文段落:**
```
それだけのことだった。
```

```
記録する必要もなかった。
```

```
何かが震えた。
```

## 終結表現のレパートリー

「それだけのことだった」のような終結表現は、本作の虚無的トーンを作る重要な技法である。
ただし、同一表現の繰り返しは陳腐化を招くため、**目的に応じた複数の表現を使い分ける**。

### 終結表現の目的

| # | 目的 | 説明 |
|---|------|------|
| 1 | **意味の剥奪** | 出来事に「意味」「重要性」を付与しない |
| 2 | **感傷の遮断** | 読者の感傷的な解釈を先回りして断つ |
| 3 | **虚無のトーン** | 宇宙的無関心（cosmic indifference）を表現 |
| 4 | **事実への還元** | 出来事を「事実」として閉じる |
| 5 | **余韻の創出** | 短文で読者に考える間を与える |

### 目的別・代替表現

#### 意味の剥奪
```
それだけのことだった。
それ以上でも以下でもなかった。
それが全てだった。
事実はそれに尽きる。
付け加えることは何もない。
```

#### 感傷の遮断
```
誰もそれを悲劇とは呼ばなかった。
悼む者はいなかった。
記録する必要もなかった。
解釈する主体が存在しなかった。
それを見届けた者はいなかった。
```

#### 虚無のトーン（宇宙的無関心）
```
宇宙は続いた。
何も変わらなかった。
恒星は燃え続けた。
時間だけが過ぎた。
銀河は回転を続けた。
```

#### 事実への還元
```
観測可能な事象はそこで終わった。
記録はそこで途切れている。
以降のデータは存在しない。
反応は停止した。
それ以降の変化は検出されなかった。
```

#### 余韻の創出（一文段落用）
```
静寂が戻った。
それきりだった。
痕跡は残らなかった。
何かが終わった。
何かが震えた。
```

### 使用上の注意

- **同一章内で同じ表現を繰り返さない**
- 章末の終結表現は特に重要。各章で異なる表現を選ぶ
- 目的が同じでも、文脈に合った表現を選択する
- 「何かが震えた」は第9章のクロージング専用として温存する

## 備考

- 冷徹さの中にも「詩情」は許容する。ただし抑制的に。
- 「美しい」「壮大な」などの形容詞は極力避ける。
- 読者に感情を「読み取らせる」のではなく、「勝手に感じさせる」文体を目指す。
