## 必須の描写要素（7つの柱）

以下は全ての小説に共通する基本的な文体ルールです。

### 1. 時間・場所の具体的な明示

**避けるべき曖昧な表現:**
```
会議が始まった。
その日の午後、主人公は作業を行った。
```

**推奨される具体的な表現:**
```
2025年6月15日、午前4時23分。
主人公の携帯電話が鳴った。

午前7時30分、主人公はバスに乗っていた。
```

**ポイント:**
- 年月日・時刻を明記
- 場所を具体的に（「会議室」ではなく「本社ビル5階の小会議室」）
- 読者が時系列を追えるように

---

### 2. 環境・雰囲気の五感描写

**避けるべきシンプルすぎる描写:**
```
部屋に入った。
外は暗かった。
```

**推奨される豊かな描写:**

| 感覚 | 例 |
|------|-----|
| **視覚** | 「会議室は想像以上に大きかった。楕円形のテーブルが中央にあり、その周りに50脚の椅子が並んでいる」 |
| **聴覚** | 「空調の音が、低く響いている」 |
| **触覚** | 「冷たい風が頬を撫でた」 |
| **嗅覚** | 「コーヒーの香りが研究室に満ちた」 |
| **味覚** | 「口の中に鉄の味が広がった」 |

**ルール:** 少なくとも視覚＋1つ以上の感覚を描写

---

### 3. 登場人物の身体状態・表情・仕草

**避けるべき記号的な描写:**
```
主人公は疲れていた。
彼女が驚いた。
```

**推奨される具体的な描写:**

```
鏡に映る自分の顔は、疲れきっていた。
目の下に隈。頬はこけている。体重は5キロ減った。

彼女の声は震えていた。
彼女がこのような声を出すのを、主人公は聞いたことがなかった。
```

**ポイント:**
- 「疲れた」ではなく「目の下に隈」など具体的に
- 感情を直接述べず、身体反応で示す
- 小さな仕草を積み重ねて人物を立体的に

---

### 4. 対話の自然な流れ

**避けるべき箇条書き的な会話:**
```
「どこにいる?」
「駅前です」
「時間は?」
「午後3時です」
```

**推奨される自然な対話:**
```
「主人公さん、お願いします」

主人公は立ち上がった。
足が、少し震えている。緊張ではない。睡眠不足だ。

主人公は壇上に上がり、リモコンを受け取った。
そして、スクリーンを操作した。

「これは昨日の分析結果です」

主人公の声は、いつもより低かった。疲労で声帯が重い。
```

**ポイント:**
- 会話の前後に動作・心理描写を挿入
- 声のトーンや話し方の特徴を描写
- 一問一答を避け、自然な間を作る

---

### 5. 短文と長文のリズム

**避けるべき単調な文体:**
```
主人公は窓を見た。外は暗い。雨が降っている。風が強い。気温は低い。
```

**推奨される変化のある文体:**
```
主人公は窓を見た。

外は暗かった。雨が、窓を叩いている。風が強い。街路樹が激しく揺れている。

気温は低く、暖房の効いた室内でも、主人公は腕を擦った。

夏のはずなのに。
```

**ポイント:**
- 短文で緊張感を作る
- 長文で情景を広げる
- 一文だけの段落で強調
- 変化をつけてリズムを生む

---

### 6. 箇条書きを自然な文章に変換

**避けるべき箇条書き的記述:**
```
施設の特徴:
- 地下5階にある
- 最新の設備が揃っている
- 24時間監視体制
- セキュリティが厳重
```

**推奨される散文化:**
```
地下5階。

エレベーターのドアが開くと、廊下には蛍光灯の冷たい光が満ちていた。
コンクリートの壁。パイプが剥き出しの天井。空調の音が、低く響いている。

最新の設備が揃っていた。壁には無数のモニター。天井には監視カメラ。
24時間、誰かが見ている。

セキュリティゲートで、IDカードをスキャンされる。
虹彩認証。指紋認証。
厳重だ。
```

**ポイント:**
- リストを文章として再構成
- 描写の中に情報を自然に織り込む
- 体験として読者に伝える

---

### 7. 内面描写の適度な挿入

**避けるべき過剰/不足:**

過剰な例:
```
主人公は思った。これは恐ろしい。とても恐ろしい。
絶望的だ。希望はない。何もできない。無力だ。
```

不足な例:
```
主人公はデータを見た。
計算した。
結果を記録した。
```

**推奨されるバランス:**
```
主人公は計算を続けた。

仮にXをYとする。
この条件で成立するには、どれだけの値が必要か。
しかし、全ての計算が、「不可能」を示した。

だが、それは現に存在している。

つまり、我々の前提が間違っている。

主人公は、自分が微笑んでいることに気づいた。

これは、恐怖ではない。
これは、興奮だ。
```

**ポイント:**
- 感情を直接述べない（「恐ろしい」→身体反応で示す）
- 思考プロセスを描写
- 矛盾する感情を描く（恐怖と興奮）
- 抑制的だが、時折感情が漏れる

## 一般的に避けるべき表現

### 1. 過度な箇条書き
```
❌ 主人公が考えたこと:
・Aは強い
・我々は弱い
・対策が必要
```

### 2. 説明的すぎる記述
```
❌ 主人公は○○である。彼は△△で働いている。
専門は□□だ。妻と娘がいる。冷静で論理的な性格だ。
```

### 3. 陳腐な比喩
```
❌ 怒りに燃える
絶望の淵に立たされた
まるで地獄のような光景
```

### 4. 感情の直接的な説明
```
❌ 非常に恐ろしかった
心が凍るような恐怖を感じた
絶望で心がいっぱいになった
```

## 挿入資料の効果的な使用

### 資料タイプ

| タイプ | 形式 | 用途 |
|--------|------|------|
| 公式記録 | 客観的、事務的 | 世界の状況を示す |
| 私的メモ | 主観的、感情的 | キャラクターの内面を深掘り |
| 報道 | 中立的 | 社会の反応を示す |
