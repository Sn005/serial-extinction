## 本作固有の文体原則

### 1. 無人称の語り

本作の最重要原則。「誰が語っているか」を示さない。

**避けるべき表現:**
```
❌ 彼らは静かに消えていった。誰にも看取られることなく。
❌ 最後の一匹が息を引き取ったとき、歴史は終わった。
❌ 私たちの祖先は〜
```

**推奨される表現:**
```
⭕ 個体数が閾値を下回った。
   遺伝的多様性が維持できなくなった。
   繁殖は停止した。
   3世代後、観測可能な個体は存在しなくなった。

⭕ 最後の個体が機能を停止したのは、
   恒星暦で[数値]年[数値]日のことだった。
   それを記録した者はいない。
   記録する必要もなかった。
```

### 2. 感情語の禁止（非知的生物）

単細胞生物、三葉虫、陸上進出者などには感情を付与しない。

**避けるべき表現:**
```
❌ 彼らは新しい環境を恐れた。
❌ 生き延びようとする意志があった。
❌ 死の瞬間、何を思っただろうか。
```

**推奨される表現:**
```
⭕ 化学勾配に反応し、鞭毛が動いた。
⭕ 個体は分裂した。それを「生きている」と呼ぶなら、それは生きていた。
⭕ 機能が停止した。それだけのことだった。
```

### 3. 知的生物の抑制的描写（ネアンデルタール人、人類、AI）

知性を持つ種でも、感傷的に描かない。

**避けるべき表現:**
```
❌ 彼らは絶望し、空を見上げた。
❌ 最後の人類は、自らの運命を悟った。
```

**推奨される表現:**
```
⭕ 道具が作られた。火が使われた。死者が埋葬された。
   それが「知性」と呼ばれるものだった。

⭕ 最後の個体が機能を停止した。
   自覚していたかどうかは記録されていない。
```

### 4. 時間スケールの明示

常に宇宙的・地質学的時間スケールを意識させる。

**推奨される表現:**
```
⭕ 宇宙誕生から97億年。

⭕ 10億年が過ぎた。
   大陸は割れ、繋がり、また割れた。
   彼らがいたことを、岩石だけが記憶している。

⭕ 3億年。
   人類の歴史の1万倍以上の時間。
   その間、彼らは海底を這い続けた。
```

### 5. 物質的連続性の描写

各章の末尾で、物質（原子、DNA、情報）の継承を描く。

**推奨される表現:**
```
⭕ 個体は分解された。
   構成していた炭素原子は海水に溶けた。
   その一部は、10億年後、別の膜に取り込まれることになる。

⭕ 彼らのDNAは、現代人の細胞の中に残っている。
   1〜4%。
   彼らは「完全には」消えていない。

⭕ 骨は地層に沈んだ。
   カルシウムは土壌に還元された。
   その上を、やがて別の足が踏むことになる。
```

## ★ 重要：物語のダイナミズム

> **このセクションは最重要ルールである。テーマへの忠実さと読者を引き込む力は両立可能であり、両立させなければならない。**

### 1. 説明ではなく描写

教科書的な説明は避け、「その場にいる」感覚を作る。

**避けるべき表現（説明調）:**
```
❌ 脂肪酸は、水中で特異な挙動を示した。
   分子の一端は水に溶けやすく、他端は水を避けた。
   この性質により、脂肪酸は自発的に集合し、二重層を形成した。
```

**推奨される表現（描写）:**
```
⭕ 脂肪酸が集まり始めた。
   一端が水を引き、他端が水を避けた。引き合い、押し合い、分子が整列した。
   層が曲がった。
   層が閉じた。
   球になった。
```

### 2. 章の構造設計

各章には明確な構造的起伏を設計する。全セクションが同じトーンにならないこと。

**構造の基本パターン:**
```
序章     ：静謐に始まる（密度：低）
展開     ：世界の確立（密度：中）
転換点   ：不穏さの導入（密度：中→高）
クライマックス：最大の変化（密度：最高）★ここに最も重みを
終結     ：静謐に終わる（密度：低）
```

**避けるべき構造:**
- 全セクションが同じ温度・密度
- クライマックスが他の事象と同列に記述される
- 起伏のない平坦な展開

### 3. 対立構造の導入

感情移入なしに緊張感を生むには、構造的な対立を設計する。

**Extinction 01の例:**
```
嫌気性生物の世界（確立）
    ↓
シアノバクテリアの出現（対立の導入）
    ↓
酸素の蓄積（緊張の高まり）
    ↓
大酸化イベント（クライマックス）
    ↓
追いやられた者たち（結末）
```

### 4. 虚無感・無常観の表現 ★最重要

感傷を排除しつつ読者の感情を揺さぶるには、**「不在」を強調する**。

**核心となる表現パターン:**

```
⭕ 誰も見ていなかった。

⭕ 誰も数えていなかった。どれだけの系統が途絶えたのか。
   どれだけの遺伝子が失われたのか。
   記録する者がいなかった。記録という概念がなかった。

⭕ 彼らには名前がなかった。
   彼ら自身が名前を持たなかったのではない。
   名前を与える者がいなかったのだ。

⭕ 停止した反応を、悼む者はいなかった。

⭕ 失われたことを知る者がいないまま、失われた。

⭕ その名前は、40億年遅かった。
   名付けられた者たちは、もういなかった。

⭕ 問いは40億年遅かった。
   問われた者たちは、答える能力を持っていなかった。40億年前も。今も。

⭕ 40億年の沈黙は、破られなかった。
   問いが生まれただけだった。答えのない問いが。

⭕ その間に40億年があった。
   誰にも見られなかった40億年が。
   誰にも名付けられなかった40億年が。
   誰にも悼まれなかった絶滅が。
   誰にも祝われなかった誕生が。
   ただ、あった。

⭕ その間に、無数の膜が震え、無数の膜が止まった。
   誰も数えなかった。
```

**不在の種類:**
- 観測者の不在（誰も見ていなかった）
- 記録者の不在（誰も記録しなかった）
- 命名者の不在（誰も名前をつけなかった）
- 悼む者の不在（誰も悼まなかった）
- 祝う者の不在（誰も祝わなかった）
- 答える者の不在（問いに答える者がいなかった）

### 5. 時間経過の表現

時間経過を単調な数字の羅列にしない。世界の変容として描く。

**避けるべき表現:**
```
❌ 1億年が過ぎた。
   10億年が過ぎた。
   20億年が過ぎた。
   30億年が過ぎた。
```

**推奨される表現:**
```
⭕ 1億年。人類の歴史を1万回繰り返す時間。
   その間に、大陸が形成され始めた。海洋プレートが沈み込み、
   マグマが上昇し、火山島が出現した。

⭕ 40億年。
   その時間を、誰も見ていなかった。
```

### 6. 反復の回避

まとめ的なセクションでも、既出内容の単純な繰り返しは避ける。新しい視点を導入する。

**避けるべきパターン:**
- 時間経過を再度列挙する
- 既に述べた事実を要約する
- 同じ構文の繰り返し

**推奨されるパターン:**
- 新しい視点（例：「観測されなかった時間」という視点）
- 異なる角度からの問いかけ
- 全体を貫く概念の深化

```
宇宙誕生から97億年。

それを「物質」と呼ぶなら、それは物質だった。

決める者がいなかった。
```

## 避けるべき表現リスト

### 感傷的表現

| 避ける | 代替 |
|--------|------|
| 悲しみに暮れた | 機能が低下した |
| 希望を持った | 行動パターンが変化した |
| 愛した | 個体間の相互作用が観測された |
| 孤独だった | 同種の個体は存在しなかった |
| 苦しんだ | 神経系に過剰な刺激が入力された |

### 目的論的表現

| 避ける | 代替 |
|--------|------|
| 生き延びるために | 結果として生き延びた |
| 進化の目的 | 進化の結果 |
| 適応しようとした | 適応した個体が残った |
| 種を残すために | 種が残った |

### 人間中心主義的表現

| 避ける | 代替 |
|--------|------|
| 人類の祖先として | 後に人類へ繋がる系統 |
| 私たちの歴史 | この惑星の歴史 |
| 知性の誕生 | 「知性」と呼ばれる機能の出現 |

## 短文と長文のリズム

本作では、短文を多用して冷徹さを強調する。

**基本パターン:**
```
[短文。事実。]
[短文。事実。]
[短文。事実。]

[やや長い文で、時間や空間のスケールを示す。]

[短文。事実。]
[短文。結論。]
```

**強調したいときの一文段落:**
```
それだけのことだった。
```

```
記録する必要もなかった。
```

```
何かが震えた。
```

## 終結表現のレパートリー

「それだけのことだった」のような終結表現は、本作の虚無的トーンを作る重要な技法である。
ただし、同一表現の繰り返しは陳腐化を招くため、**目的に応じた複数の表現を使い分ける**。

### 終結表現の目的

| # | 目的 | 説明 |
|---|------|------|
| 1 | **意味の剥奪** | 出来事に「意味」「重要性」を付与しない |
| 2 | **感傷の遮断** | 読者の感傷的な解釈を先回りして断つ |
| 3 | **虚無のトーン** | 宇宙的無関心（cosmic indifference）を表現 |
| 4 | **事実への還元** | 出来事を「事実」として閉じる |
| 5 | **余韻の創出** | 短文で読者に考える間を与える |

### 目的別・代替表現

#### 意味の剥奪
```
それだけのことだった。
それ以上でも以下でもなかった。
それが全てだった。
事実はそれに尽きる。
付け加えることは何もない。
```

#### 感傷の遮断
```
誰もそれを悲劇とは呼ばなかった。
悼む者はいなかった。
記録する必要もなかった。
解釈する主体が存在しなかった。
それを見届けた者はいなかった。
```

#### 虚無のトーン（宇宙的無関心）
```
宇宙は続いた。
何も変わらなかった。
恒星は燃え続けた。
時間だけが過ぎた。
銀河は回転を続けた。
```

#### 事実への還元
```
観測可能な事象はそこで終わった。
記録はそこで途切れている。
以降のデータは存在しない。
反応は停止した。
それ以降の変化は検出されなかった。
```

#### 余韻の創出（一文段落用）
```
静寂が戻った。
それきりだった。
痕跡は残らなかった。
何かが終わった。
何かが震えた。
```

### 使用上の注意

- **同一章内で同じ表現を繰り返さない**
- 章末の終結表現は特に重要。各章で異なる表現を選ぶ
- 目的が同じでも、文脈に合った表現を選択する
- 「何かが震えた」は第9章のクロージング専用として温存する

## ★ 表現の多様性と厚み（重要）

> **Extinction 04執筆時のフィードバックに基づく追加ルール。同一パターンの繰り返しは陳腐化を招き、読者の没入を妨げる。**

### 1. 終結表現の分散

**ルール：同一の終結表現を章内で2回以上使用しない**

特に以下の表現は陳腐化しやすいため、代替表現を積極的に使用すること：

| 陳腐化しやすい表現 | 代替表現の例 |
|------------------|-------------|
| それだけのことだった | 付け加えることは何もない / それ以上でも以下でもなかった / 事実はそれに尽きる |
| 誰も見ていなかった | 目撃する者は存在しなかった / 記録する者がいなかった / その瞬間を捉えた者はいなかった |
| 〜を知らなかった | 認識する能力がなかった / 〜という概念が存在しなかった / 認識していなかった |
| 意図はなかった | 方向性はなかった / 設計という概念が存在しなかった / 目的もなかった |
| 確証はない | 証明する方法はない / 確かなのは結果だけだった / 詳細は不明確だ |

### 2. 冗長な反復の削減

**ルール：同じ構文パターンを連続して3回以上使用しない**

**避けるべきパターン：**
```
❌ 〜は消えた。
   〜は消えた。
   〜は消えた。
   〜は消えた。
```

**推奨されるパターン：**
```
⭕ 〜は消えた。
   〜も姿を消した。
   〜の大部分が途絶えた。
```

### 3. 描写の厚み

**ルール：各セクションに具体的な詳細（数値、固有名詞、物理的描写）を含める**

**薄い描写（避ける）：**
```
❌ 浅瀬に生物がいた。環境は厳しかった。
```

**厚みのある描写（推奨）：**
```
⭕ 水温は摂氏30度を超えることもあった。温かい水には、酸素が溶けにくかった。
   アーキオプテリス。高さ30メートルに達する最初の「木」。
   その落ち葉や有機物が水中に流れ込んでいた。
```

**厚みを出す要素：**
- 具体的な数値（温度、サイズ、距離、年代）
- 固有名詞（種名、地名、人名）
- 物理的・化学的プロセスの詳細
- 比較対象（「〜の何倍」「現代の〜に相当」）

### 4. 科学用語の正確な表記

**ルール：専門用語は正確に表記し、文字化けや誤字を防ぐ**

執筆後に以下をチェック：
- [ ] 化学式・単位の正確性
- [ ] 学名のイタリック表記の一貫性
- [ ] 専門用語の表記ゆれがないか
- [ ] 文字化けがないか（特に「石化」「石灰岩」などの漢字）

---

## 実践チェックリスト（追加項目）

### 表現の多様性
- [ ] 「それだけのことだった」系の表現が章内で1回以下か
- [ ] 「誰も見ていなかった」系の表現が過度に連続していないか
- [ ] 同じ構文パターンが3回以上連続していないか
- [ ] 終結表現が章全体で分散されているか

### 描写の厚み
- [ ] 各セクションに具体的な数値や固有名詞が含まれているか
- [ ] 物理的・化学的プロセスが詳細に描写されているか
- [ ] 比較対象を使って時間や規模を実感させているか

---

## 備考

- 冷徹さの中にも「詩情」は許容する。ただし抑制的に。
- 「美しい」「壮大な」などの形容詞は極力避ける。
- 読者に感情を「読み取らせる」のではなく、「勝手に感じさせる」文体を目指す。
