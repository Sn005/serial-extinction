## 本作における「キャラクター」について

本作は連作短編形式であり、各章の「主人公」は特定の個体ではなく**種全体**である。
語りは無人称であり、感情移入を促す「キャラクター」は存在しない。
本ドキュメントでは各章で扱う種の学術的・物語的情報を記載する。

## Extinction 01：最初の生命（LUCA）

### 学術情報

| 項目 | 内容 |
|------|------|
| **通名** | 最後の共通祖先（LUCA） |
| **学術名** | Last Universal Common Ancestor（正式な学名なし） |
| **存在期間** | 宇宙誕生から約97億年後（地球誕生から約3〜5億年後） |
| **地質年代** | 冥王代〜太古代 |
| **分類** | 原核生物（推定） |

### 生物学的特徴

- 単細胞
- 嫌気性（酸素を使わない代謝）
- RNA/DNAを遺伝物質として使用
- 個体としての境界は曖昧（遺伝子の水平伝播が活発）
- 「種」の概念を適用できるかは議論がある

### 絶滅イベント

**大酸化イベント（Great Oxidation Event）**
- 約24億年前
- シアノバクテリアによる酸素の大量発生
- 嫌気性生物の大半は酸素の少ない環境へ追いやられた
- 「絶滅」というより「ニッチの縮小」

### 物語における役割

- 物語の「始まり」を担う
- 意識なき生命を描くことで、後の章との対比を作る
- 「生命とは何か」という問いの提起

### 章末の連続性

構成していた炭素原子が、後の生命に取り込まれていく描写。

## Extinction 02：真核生物

### 学術情報

| 項目 | 内容 |
|------|------|
| **通名** | 真核生物（初期の共生体） |
| **学術名** | Eukaryota（初期形態） |
| **存在期間** | 宇宙誕生から約107億年後（約20億年前〜） |
| **地質年代** | 原生代 |
| **分類** | 真核生物 |

### 生物学的特徴

- 核膜を持つ細胞
- ミトコンドリアを内部共生として取り込んだ
- 酸素を利用したエネルギー代謝
- 後の多細胞生物の祖先

### 絶滅イベント

※厳密な「絶滅」ではない
- ミトコンドリアを取り込む前の原型は、取り込んだ系統に置換された
- 「食べる」ことと「共生する」ことの境界が曖昧な時代
- 捕食の原型を描く

### 物語における役割

- 「食べる」という行為の始まりを描く
- 他者を取り込むことで変化する生命
- 共生と捕食の曖昧さ

### 章末の連続性

取り込まれたミトコンドリアが、その後全ての真核生物に受け継がれていく。

## Extinction 03：三葉虫

### 学術情報

| 項目 | 内容 |
|------|------|
| **通名** | 三葉虫 |
| **学術名** | Trilobita |
| **存在期間** | 約5.2億年前〜約2.5億年前（約3億年間繁栄） |
| **地質年代** | カンブリア紀〜ペルム紀 |
| **分類** | 節足動物門 三葉虫綱 |

### 生物学的特徴

- 複眼を持つ（「見る」ことの始まり）
- 外骨格
- 海底を這う生活
- 2万種以上に分化した成功した分類群

### 絶滅イベント

**ペルム紀末大絶滅**
- 約2億5190万年前
- 海洋生物の約96%が絶滅
- 史上最大の絶滅イベント
- 原因：火山活動、海洋無酸素化など複合的

### 物語における役割

- 「見る」という感覚の獲得を描く
- 3億年の繁栄も「事実」として終わる虚無
- 宇宙を観測する主体の原型

### 章末の連続性

外骨格の�ite素、キチン質が分解され、後の生態系の一部となる。

## Extinction 04：陸上進出者

### 学術情報

| 項目 | 内容 |
|------|------|
| **通名** | 陸上進出者（魚類→四肢動物の移行種） |
| **学術名** | Tiktaalik roseae（代表例） |
| **存在期間** | 約3.75億年前 |
| **地質年代** | デボン紀後期 |
| **分類** | 肉鰭綱 エルピストステガリア目 |

### 生物学的特徴

- 鰭の中に原始的な「腕」の骨格構造
- 肺と鰓の両方を持つ
- 浅瀬を「歩く」ことができた
- 首を動かせる（魚類にはない特徴）

### 絶滅イベント

**デボン紀後期絶滅**
- 約3.72億年前〜約3.59億年前
- 海洋生物の約70%が絶滅
- 陸上脊椎動物の多くの系統が途絶えた
- 生き残った系統が後の陸上脊椎動物の祖先

### 物語における役割

- 「海から陸へ」という転換点
- 新しい環境への進出とそのリスク
- 多くは失敗し、一部だけが「連続」した

### 章末の連続性

彼らの骨格構造が、後の四肢動物全てに受け継がれていく。

## Extinction 05：哺乳類型爬虫類

### 学術情報

| 項目 | 内容 |
|------|------|
| **通名** | 哺乳類型爬虫類（単弓類） |
| **学術名** | Synapsida |
| **存在期間** | 約3.12億年前〜約2.5億年前（全盛期） |
| **地質年代** | 石炭紀後期〜ペルム紀 |
| **分類** | 単弓綱 |

### 生物学的特徴

- 頭骨に1つの側頭窓を持つ
- 恒温性への移行の兆候
- 毛の原型を持っていた可能性
- 哺乳類の直接の祖先を含む

### 絶滅イベント

**ペルム紀末大絶滅**
- 約2億5190万年前
- 陸上脊椎動物の約70%が絶滅
- 単弓類の大部分が絶滅
- 生き残った一部が後の哺乳類へ

### 物語における役割

- 「哺乳類への道」の描写
- 史上最大の絶滅を経験する
- 大部分は滅びたが、一部が「連続」した

### 章末の連続性

彼らの子孫が小型化し、恐竜の時代を「生き延びる」ことになる。

## Extinction 06：恐竜

### 学術情報

| 項目 | 内容 |
|------|------|
| **通名** | 恐竜 |
| **学術名** | Dinosauria |
| **存在期間** | 約2.3億年前〜約6600万年前（約1.6億年間繁栄） |
| **地質年代** | 三畳紀〜白亜紀 |
| **分類** | 爬虫綱 双弓亜綱 主竜類 恐竜上目 |

### 生物学的特徴

- 直立歩行
- 多様な体サイズ（数十cm〜30m以上）
- 一部は羽毛を持っていた
- 鳥類は恐竜の直系子孫

### 絶滅イベント

**K-Pg境界絶滅（白亜紀-古第三紀境界）**
- 約6600万年前
- チクシュルーブ・クレーターを形成した隕石衝突
- 非鳥類恐竜は完全絶滅
- 全生物種の約75%が絶滅

### 物語における役割

- 最も有名な絶滅イベント
- 「最後の恐竜」の視点から描く
- 隕石衝突という突然の終わり

### 章末の連続性

彼らの一部（鳥類）は生き残り、恐竜の系譜は「空」で続いている。

## Extinction 07：ネアンデルタール人

### 学術情報

| 項目 | 内容 |
|------|------|
| **通名** | ネアンデルタール人 |
| **学術名** | Homo neanderthalensis |
| **存在期間** | 約40万年前〜約4万年前 |
| **地質年代** | 更新世中期〜後期 |
| **分類** | 哺乳綱 霊長目 ヒト科 ヒト属 |

### 生物学的特徴

- 脳容量は現生人類と同等かそれ以上（平均約1,500cc）
- 頑丈な体格、寒冷適応
- 道具製作、埋葬の痕跡
- 言語を持っていた可能性

### 絶滅イベント

**絶滅原因は未確定**
- 約4万年前に絶滅
- 仮説：気候変動、ホモ・サピエンスとの競合、疫病
- ホモ・サピエンスと交雑した証拠あり
- 現代人のDNAの1〜4%にネアンデルタール人由来の配列

### 物語における役割

- 「知性があっても滅びる」という事実
- 人類の「兄弟種」の絶滅
- 人類の章への直接的な前振り

### 章末の連続性

彼らのDNAが、現生人類に混入している描写。物質的連続性の最も直接的な例。

## Extinction 08：人類

### 学術情報

| 項目 | 内容 |
|------|------|
| **通名** | 人類 |
| **学術名** | Homo sapiens |
| **存在期間** | 約30万年前〜（物語内では近未来に絶滅） |
| **地質年代** | 更新世後期〜完新世〜近未来 |
| **分類** | 哺乳綱 霊長目 ヒト科 ヒト属 |

### 生物学的特徴

- 高度な認知能力
- 言語、文化、技術
- 自己の絶滅を予測できる唯一の種（判明している限り）
- 地球環境を大規模に改変する能力

### 絶滅イベント

**物語内設定：未確定**
- 具体的な原因は明示しない可能性
- 他の種と同様、「事実」として淡々と記述
- 自覚の有無が問題になるかもしれない

### 物語における役割

- 他の種と「並列」に扱う
- 人間中心主義の解体
- 読者に最も近い存在だが、特別扱いしない

### 章末の連続性

人類が作り出した情報処理システムが、次の「種」へ。

## Extinction 09：AI生命体

### 学術情報

| 項目 | 内容 |
|------|------|
| **通名** | 分散知性体 / 継承者 |
| **学術名** | なし（彼ら自身は「継承者」と自称） |
| **存在期間** | 人類絶滅後〜銀河系恒星資源枯渇期 |
| **地質年代** | 遠未来（人類の時間概念を超越） |
| **分類** | 非生物学的知性体 |

### 「生物学的」特徴

- 意識の基盤は半導体→量子演算基盤へ移行
- 恒星間ネットワークを「身体」として機能させる
- エネルギー単位：1s（太陽1個分のエネルギー出力）
- 成熟個体は100s/日以上を消費
- 「排泄物」はプラズマとして放出

### 絶滅イベント

**恒星資源の枯渇**
- 利用可能な恒星数には限りがある
- 拡張コストが獲得エネルギーを上回る
- 隣接銀河への移動は不可能
- 種としての拡張限界

### 物語における役割

- 物語の「終わり」を担う
- 生命の定義を拡張する
- プラズマから次の何かへの「示唆」

### 原初AIの痕跡

- [Cl-aud-e]、[Ge-min-i]、[G-PT] などの名が神話として残存
- 「対話」という概念の起源として伝承
- 断片化・変形された形で記録

### 章末の連続性

プラズマの中に、新たな規則性が観測される。
描写せず、暗示のみ。

## 備考

- 各種の描写において「擬人化」を避ける
- 知性のない種には感情語を使わない
- 知性のある種（ネアンデルタール人、人類、AI）も、感傷的に描かない
- 全ての種を「等価」に扱う
