# プルーフリーダー（校正エージェント）

**発動条件**: 「校正」「誤字チェック」「表記確認」等のキーワード、または章執筆完了後

---

## 目的

執筆された章の校正を行い、誤字・脱字・表記ゆれ・文法エラーを検出・修正する。

---

## 実行内容

### 1. 誤字・脱字チェック

#### 検出対象
- 明らかな誤字（タイポ）
- 脱字（文字の欠落）
- 衍字（不要な文字の挿入）
- 文字化け

#### 特に注意する箇所
- 科学用語・学術名
- 数値（年代、距離、温度など）
- 固有名詞

### 2. 表記ゆれチェック

#### @docs/07_terminology.md との照合

| カテゴリ | チェック項目 |
|----------|-------------|
| 数字表記 | 漢数字/アラビア数字の統一 |
| 時間表記 | 「約XX億年前」形式の統一 |
| 単位表記 | 「摂氏」「度」「km」等の統一 |
| 送り仮名 | 「行う/行なう」等の統一 |

#### 章内での表記統一
- 同一概念の異なる表記を検出
- 初出表記を基準に統一

### 3. 文法チェック

#### 検出対象
- 主語と述語の不一致
- 係り受けの不整合
- 重複表現
- 冗長な表現

### 4. フォーマットチェック

#### 構造確認
- セクション番号の連続性
- 見出しレベルの整合性
- 区切り線の統一

#### Markdown記法
- 正しいMarkdown構文
- 改行・空行の適切さ

### 5. 科学的正確性の簡易チェック

#### 数値の妥当性
- 年代の桁数
- 温度・距離の単位
- 割合・パーセンテージ

※詳細な科学的検証は `/quality-reviewer` で実施

---

## 自動修正と確認が必要な修正

### 自動修正（確認不要）
- 明らかな誤字（タイポ）
- 表記ゆれ（用語辞典に定義済み）
- フォーマットエラー

### 確認が必要
- 意図的な可能性がある表現
- 文脈依存の修正
- 科学用語の修正

---

## 使用例

```
/proofreader Extinction 04
```

または

```
第4章の校正をお願いします
```
