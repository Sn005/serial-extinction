# チャプターライター

**発動条件**: 「章執筆」「執筆開始」「書いて」等のキーワード、またはアウトライン承認後

---

## 目的

アウトラインに基づき、文体ガイドラインを遵守しながら章を執筆する。

---

## 実行内容

### 1. 事前確認

執筆開始前に以下を確認：

- [ ] `/chapter-outliner` のアウトラインが存在するか
- [ ] 前章との接続ポイントを確認
- [ ] 回収すべき伏線を確認
- [ ] 使用する終結表現・虚無表現を確認

### 2. 執筆プロセス

#### Phase 1: 冒頭ヘッダーの作成

```markdown
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## Extinction XX

**■ 通名**
[種の一般名]

**■ 学術名**
[学名]
※[補足があれば]

**■ 存在期間**
宇宙誕生から約XX億年後
（地球誕生から約XX億年後 / 約XX年前）
── [地質年代]

**■ 概要**
[3〜5行の学術的概要]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

#### Phase 2: セクション単位の執筆

各セクションを順番に執筆：

1. **セクション目的の確認**: アウトラインの目的を再確認
2. **描写要素の組み込み**: 科学的事実、環境描写、時間表現
3. **文体ルールの適用**:
   - 無人称の維持
   - 感情語の禁止（非知的生物）
   - 短文と長文のリズム
   - 「不在」による虚無感の表現
4. **終結表現の適用**: 事前選定した表現を使用

#### Phase 3: Epilogue（物質的連続性）

- 物質的連続性の描写
- 次章への接続を暗示
- 章全体を閉じる表現

### 3. 執筆中の品質チェック

#### リアルタイムチェック項目
- [ ] 説明調になっていないか
- [ ] 同一表現の繰り返しがないか
- [ ] 科学的事実が正確か
- [ ] 時間スケールが明示されているか

### 4. 文体ガイドライン遵守

#### 必須ルール（@docs/05_style-guide.md）

**無人称の語り**
```
❌ 彼らは静かに消えていった。
⭕ 個体数が閾値を下回った。繁殖は停止した。
```

**感情語の禁止（非知的生物）**
```
❌ 彼らは新しい環境を恐れた。
⭕ 化学勾配に反応し、鞭毛が動いた。
```

**虚無感の表現**
```
⭕ 誰も見ていなかった。
⭕ 記録する者がいなかった。
⭕ 悼む者はいなかった。
```

**時間スケールの表現**
```
⭕ 3億年。人類の歴史の1万倍以上の時間。
```

